            flutter研发总结

            X-Man 2019-03-09 flutter

            # 基础篇

            • flutter实战(书籍) book.flutterchina.club
              可以以此为基础进行入门.里面有常用的组件分类等知识
            • 阿里 flutter-go
              阿里出品自然不会差到哪去,可以阅读此源码进行深入学习
            • github:
              • Flutter-learning
                也是一个资源汇总库来的
                Flutter安装和配置，Flutter开发遇到的难题，Flutter示例代码和模板，Flutter项目实战，Dart语言学习示例代码
            • flutter实战
              • 一、Dart语言和Flutter基础
              • 二、 快速开发实战篇
              • 三、 打包与填坑篇
              • 四、 Redux、主题、国际化
              • 五、 深入探索
              • 六、 深入Widget原理

            # 社区篇

            • flutter中文网 flutterchina.club
              比较健全的社区
            • flutter社区中文资源 flutter-io.cn
              比较健全的社区
            • 掘金社区 juejin.im
              非常好的聚集地

            # 官网的第三方插件

            pub.dartlang.org
            第三方库直接在此搜索

            # 实战篇

            • github:
              • FlutterDemo
                日更的FlutterDemo合集，今天你fu了吗
                里面有各种第三方插件demo等,资源很全
              • Flutter UI Kit
                一个Flutter UI组件库
              • Flutter完整项目wanandroid
                BLoC、RxDart 、国际化、主题色、启动页、引导页，拥有较好的项目结构&比较规范的代码！
              • flutter_weixin
                一个高仿微信的案例
              • flutter_gank
                干货网,很好的借鉴项目,注释等都有.不定时更新

            # 博文篇

            • 状态管理
              • 状态管理特别篇——Provide
                状态管理中的一种
              • 阿里宣布开源 Flutter 应用框架 Fish Redux
            • Flutter 布局控件完结篇
              对布局的讲解比较深入.各种理论知识
            • Flutter 一些常用库
            • Flutter实战详解--高仿好奇心日报
              有着非常好的讲解.寻寻渐进.值得一看
            • 路由管理
              • 自定义路由切换动画
              • Flutter进阶：路由、路由栈详解及案例分析
                Navigator最好的讲解.利用场景加上图解.剖析的很透彻

            • 一个很棒的等待动画库:Spinkit
